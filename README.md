# dbd-ocr



### Key

`[GENS LEFT] [[STATUS][HOOKS][ISOBSESSION]]*4`

Example: `1 D0O C0N I0N K0N`

#### Gens left
- 1-5 - Gens left
- 0 - Gates powered

#### Status
- H - Healthy
- E - Escaped
- I - Injured
- D - Disconnected
- S - Sacrificed
- C - Catched (hooked)
- K - Killed

#### Hooks
0 if never hooked, or unknown (killer).

1 or 2 if hooked.

#### IsObsession
- O - Obsession
- N - Not Obsession
